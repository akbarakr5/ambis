# Assets Placeholder

The following asset files are required but not included:

- `icon.png` (1024x1024) - App icon
- `adaptive-icon.png` (1024x1024) - Android adaptive icon
- `favicon.png` (48x48) - Web favicon

You can:
1. Use default Expo assets by running: `npx expo install expo-asset`
2. Create your own icons
3. Use placeholder images temporarily

For now, the app will work without these files, but you'll see warnings.
